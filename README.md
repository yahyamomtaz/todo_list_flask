# todo_list_flask

A simple todo list app built with Flask and SQLite.

## Features

- Add a todo item
- Mark a todo item as complete
- View all todos
- View incomplete todos
- View complete todos

## Installation & Run

```console
pip install Flask Flask-SQLAlchemy Flask-Migrate
```

```console
python run.py create_db
```

```console
python run.py run
```
